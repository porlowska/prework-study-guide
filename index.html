  • REMEMBER HTML code is CASE- SENSITIVE
  • HEADER
    • The head element contains information about the webpage.
    • It helps the browser to render the page correctly.
    • Elements included in head are NOT visible when user looks at the webpage.
    • It is a Parent Element
    • Contais Metadata
    • Examples of metadata tags - CHILD ELEMENTS : title, styles, base, link, meta, script.
  • BODY
    • The body element represents the visible content shown to the user.
    • semantic elements - the name of element describes its content and the role it plays on a webpage. It also makes sure that the page is accessible.
      • h1, h1, h3, h4, h5, h6 - level of heading.
      • p - paragraph
      • lists: ul - unordered list/bullet points and ol - ordered list/number, and li - list item
      • img - contains infor about image (img: scr, alt, width, height)
      • a href - hyperlinked text
    • non semantic element - mosty used for sutucturals problems only.
      • div - to create division in the page
